# Use with maven

#### Add repository
```xml
<repository>
    <snapshots>
        <enabled>true</enabled>
    </snapshots>
    <id>divroll</id>
    <name>Divroll Maven Packages</name>
    <url>https://maven.pkg.github.com/divroll</url>
</repository>
```

#### Add dependency

```xml
<dependency>
   <groupId>com.divroll</groupId>
   <artifactId>json-gwt</artifactId>
   <version>0-SNAPSHOT</version>
</dependency>
```
