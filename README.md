# Use with maven

#### Add repository
```aidl
<repository>
   <snapshots>
      <enabled>false</enabled>
   </snapshots>
   <id>bintray-divroll-oss</id>
   <name>bintray</name>
   <url>https://dl.bintray.com/divroll/oss</url>
</repository>
```

#### Add dependency

```aidl
<dependency>
   <groupId>com.divroll</groupId>
   <artifactId>json-gwt</artifactId>
   <version>0</version>
</dependency>
```
